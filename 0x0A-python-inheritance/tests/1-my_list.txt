This module defines a class that extends native list class.

=====
Usage
=====

``my_list.append(value)`` append new value to my_list

	>>> MyList = __import__('1-my_list').MyList
	>>> my_list = MyList()
	>>> my_list.append(1)
	>>> my_list.append(4)
	>>> my_list.append(2)
	>>> my_list.append(3)
	>>> my_list.append(5)

``print(my_list)`` print my_list elements

	>>> my_list = MyList()
	>>> my_list.append(1)
	>>> my_list.append(4)
	>>> my_list.append(2)
	>>> my_list.append(3)
	>>> my_list.append(5)
	>>> print(my_list)
	[1, 4, 2, 3, 5]

``my_list.print_sorted()`` Print sorted version of my_list

	>>> my_list = MyList()
	>>> my_list.append(1)
	>>> my_list.append(4)
	>>> my_list.append(2)
	>>> my_list.append(3)
	>>> my_list.append(5)
	>>> my_list.print_sorted()
	[1, 2, 3, 4, 5]

We can pass list of element to MyList as the native list

		>>> my_list = MyList([1, 4, 5, 3, 2])
		>>> print(my_list)
		[1, 4, 5, 3, 2]
